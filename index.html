Form Styling

Input

Input boxes can be any width. Text inside must be dark colour.

Text Area

Text area boxes follow similar rules as the input text boxes. Text inside must be dark

Select

All font inside must be bold and dark on a white background.

Radio

Labels for forms must be dark and clear. Radio buttons must use a dark and light colour.

Colour palettes

Emerald is used to to show that things have been approved, confirmed and accepted.
Red is used for rejection e.g could not sign in.

Sets

Rules for colours are as follows. These are the colour sets that can be used omn a page. You are are allowed to complimenting solid block colours. Font on these colours need to be white for sufficient contrast. You are allowed two neutral colours that compliment the solid colours and achromatic grey. You are also allowed to black for text at any point in the page.
